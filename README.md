# 📨 Telegram Aggregator

Userbot который пересылает сообщения из нужных каналов в один канал-агрегатор.

---

## 🚀 Установка и запуск (пошагово)

### Шаг 1 — Получи Telegram API ключи

1. Зайди на [my.telegram.org](https://my.telegram.org) **под своим вторым аккаунтом**
2. Нажми **"API development tools"**
3. Заполни форму (App title и Short name — можно написать что угодно, например "MyAggregator")
4. Скопируй **App api_id** (число) и **App api_hash** (длинная строка)

---

### Шаг 2 — Узнай ID своего канала-агрегатора

1. Создай новый канал в Telegram (или используй существующий)
2. Добавь бота [@userinfobot](https://t.me/userinfobot) как администратора канала
3. Перешли любое сообщение из канала боту @userinfobot
4. Он пришлёт ID — это будет число вида `-1001234567890`
5. Скопируй это число

---

### Шаг 3 — Настрой проект

```bash
# Установи зависимости
pip install -r requirements.txt

# Скопируй файл с примером настроек
cp .env.example .env
```

Открой файл `.env` и заполни своими данными:
```
API_ID=12345678                        ← вставь свой api_id
API_HASH=abcdef...                     ← вставь свой api_hash
PHONE_NUMBER=+79991234567              ← номер второго аккаунта
AGGREGATOR_CHANNEL_ID=-1001234567890   ← ID твоего канала
```

---

### Шаг 4 — Добавь каналы для слежки

Открой `config.py` и добавь username каналов которые хочешь агрегировать:

```python
SOURCE_CHANNELS = [
    "username_канала_1",
    "username_канала_2",
    "username_канала_3",
]
```

---

### Шаг 5 — Запусти!

```bash
python main.py
```

**При первом запуске:**
- Введи номер телефона (если спросит)
- Введи код из Telegram
- Готово! Сессия сохранится в файл `aggregator_session.session`

При следующих запусках — авторизация не нужна.

## ❓ Частые вопросы

**Программа упала с ошибкой — что делать?**
Посмотри текст ошибки в консоли. Чаще всего причина:
- Неверный API_ID или API_HASH → перепроверь .env
- Неверный ID канала → перепроверь AGGREGATOR_CHANNEL_ID
- Бот не является членом канала-источника → подпишись на канал через второй аккаунт

**Как остановить бота?**
Нажми `Ctrl+C` в консоли.

**Файл aggregator_session.session — что это?**
Сохранённая сессия. Не удаляй его, иначе придётся логиниться заново.
